# 杨辉三角

## Description
输入一个数字n（1<=n<=10), 不在范围内的n输出“Out Of Range.\n"
输出杨辉三角的前n行
 

## Sample Input
6  

 

## Sample Output
1

1&emsp;1

1&emsp;2&emsp;1

1&emsp;3&emsp;3&emsp;1

1&emsp;4&emsp;6&emsp;4&emsp;1

1&emsp;5&emsp;10&emsp;10&emsp;5&emsp;1

## Sample Input
16  

 

## Sample Output
 
Number 16 is Out Of Range.
