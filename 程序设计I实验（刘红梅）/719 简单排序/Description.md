# 简单排序

## Description

有一列数量为n的数列，请你按照升序给他们排序，然后输出。

i输入数据：第一行为一个整数n（3<n<=100），第二行为n个整数(不超过2^31)，是待排序的数列。

输出数据：一行n个整数，以空格隔开，是排过的序列，最后一个数没有空格。

 

## Sample input：
```
5
50 10 20 30 40
```

## Sample output：
```
10 20 30 40 50
```
