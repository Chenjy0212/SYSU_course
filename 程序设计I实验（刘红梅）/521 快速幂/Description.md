# 快速幂

## Description:

given a b and p, output (a^b) % p   (2<=a<=100, 0<=b<=1000000000,  3<=p<=10000)

## Input
3 3 10

## Output
7

## Hint:
编写函数：

int quick_power(int, int, int);
